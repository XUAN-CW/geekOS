bochs的安装

静虑凡尘 2018-08-04 17:26:42 3395 收藏 7
版权

环境

Ubuntu 16.04 + bochs 2.6.9 + nasm

直接获取

可以直接使用命令下载：

        sudo apt-get install bochs

下载bochs

https://sourceforge.net/projects/bochs/files/bochs/

选择最新的版本

在这个网站下载bochs 2.6.9.tar.gz压缩包

安装依赖环境：

      sudo apt-get install build-essential xorg-dev libgtk2.0-dev

配置参数打开调试功能开关

    ./configure --enable-debugger --enable-disasm

编译

  make

 

安装

sudo make install 

输入命令

  bochs

如果能显示以下代码表示安装成功
