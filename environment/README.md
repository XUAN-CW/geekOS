# 创建虚拟机

侧边栏**工具** → **新建** → 输入名字、选择放置虚拟机的文件夹 → 版本选择 `Ubuntu (32-bit)` 

剩下的直接无脑 next

## 安装

### install

直接双击 Install，除了账号密码，剩下的一路 next 即可。然后就是静静等待，如果卡住了，多按几次回车

![image-20210511162443356](image/image-20210511162443356.png)
